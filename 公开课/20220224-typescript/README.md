## 安装 typescript
```bash
npm install -g typescript
```

> 在 Windows 系统上可能会遇到 *“无法加载文件C:\Users\Administrator\AppData\Roaming\npm\tsc.ps1，因为在此系统上禁止运行脚本。”*

> 解决办法：管理员身份运行windows powershell 输入 *set-ExecutionPolicy RemoteSigned* 选择 A 或者 Y，回车
